Jobcoin API

Lets you look up address balances, transaction histories, and send Jobcoins. You probably won’t need all these API endpoints to solve the given coding challenge. All endpoints allow a callback parameter for JSONP.

Addresses 

Address Info 

Address info
GET
http://jobcoin.gemini.com/pout-jersey/api/addresses/{address}

Get the balance and list of transactions for an address.

Notes:

  • This will never return 404 - for an unused address, it will return a balance of 0 and an empty list of transactions.
  • Numbers will be returned as strings to preserve precision.
  • Some transactions lack a “fromAddress” - those are ones where Jobcoins were created from the UI.
  • Parameters
  • address
    string (required) Example: BobsAddress
  • Response  200Toggle
  • Body
    {
    "balance": "20.25",
    "transactions": [
    {
    "timestamp": "2014-04-22T13:10:01.210Z",
    "toAddress": "BobsAddress",
    "amount": "50.35"
},
    {
    "timestamp": "2014-04-23T18:25:43.511Z",
    "fromAddress": "BobsAddress",
    "toAddress": "AlicesAddress",
    "amount": "30.1"
}
    ]
}

Send Jobcoins
POST
http://jobcoin.gemini.com/pout-jersey/api/transactions

Send Jobcoins from one address to another.

  • Parameters
  • fromAddress
    string (required) Example: BobsAddress

    The address sending the Jobcoins

    toAddress
    string (required) Example: AlicesAddress

    The address receiving the Jobcoins

    amount
    string (required) Example: 30.1

    The number of Jobcoins to send, as a string.

  • Response  200Toggle
  • Body
    {
    "status": "OK"
}
  • Response  422Toggle
  • Body
    {
    "error": "Insufficient Funds"
}
